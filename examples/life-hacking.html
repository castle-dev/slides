  • Managing Windows

    Open new window = ctrl + n

    Previous window = alt + tab

  • Managing Tabs

    Open new tab = ctrl + t

    Close current tab = ctrl + w

    Next tab = ctrl + tab

    Previous tab = ctrl + shift + tab

  • Navigation

    Scroll down = space

    Scroll up = shift + space

    Go back = alt + left

    Go forward = alt + right

  • Entering Data

    Jump to next input = tab

    Jump to previous input = shift + tab

    Jump cursor by word = ctrl + left/right

  • Manipulating Text

    Cut = ctrl + x

    Copy = ctrl + c

    Paste = ctrl + v

    Paste without formatting = ctrl + shift + v

  • Googling

    Focus address bar = ctrl + l

    Search from the address bar

    Use tab and arrows to select results

    Start with "define:" to get definitions
